监控插件 - 使用说明

✨ 使用方式
只需要在 script 标签的 src 属性中添加参数，插件就会自动初始化，无需任何额外代码！

使用方法

直接在 script 标签的 src 属性中添加参数：

<script src="https://your-domain.com/plugin/monitoring.js?key=你的项目Key&apiUrl=你的API地址"></script>

完整示例：

<script src="https://example.com/plugin/monitoring.js?key=0346c20e5580a8d83b0372089bbfdd8d&apiUrl=http://web.yuwb.cn"></script>
⚠️ 重要提示：
  • key 或 projectKey：项目Key（必填）
  • apiUrl：API服务器地址（必填）
  • 两个参数都必须提供，否则插件无法正常工作

参数说明

  • key 或 projectKey：项目Key（必填），在前端面板创建项目后获取
  • apiUrl：API服务器地址（必填）
    • 开发环境：http://localhost:3002
    • 生产环境：http://web.yuwb.cn 或 https://web.yuwb.cn

测试示例

下面是一个完整的测试页面示例：

<!-- 监控插件 - 使用示例 --> <!-- 请将下面的 key 和 apiUrl 替换为你的实际值 --> <script src="./monitoring.js?key=your-project-key&apiUrl=http://localhost:3002"></script>

注意事项

  • 插件会自动从 script 标签的 src 参数中读取配置并初始化
  • 无需调用任何 JavaScript 代码，插件会自动开始工作
  • 如果配置不完整，插件会在控制台输出错误信息
  • 确保后端服务正常运行，并且允许跨域请求（CORS）
